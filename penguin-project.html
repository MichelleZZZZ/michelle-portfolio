About the
project

A Website was built from scratch with smooth animations which aims to call on protecting wild animals like Yellow-eyed Penguin. I committed to making this project storytelling and interactive so I iterated SVG animations on my website using GreenSock to achieve the visual effect.

UI / UX / HTML+CSS / Sass / jQuery / GreenSock

UX & UI
Design Process

Information Hierarchy

I collected all contents from Yellow-Eyed Penguin Trust website(http://www.yellow-eyedpenguin.org.nz/ )and rearranged the information hierarchy.

Design a story

I believe good design is telling a story. I divided the information hierarchy to four parts: Who are Yellow Eyed Penguins and what situation they face, What is Yellow Eyed Penguin Trust, the works of the Trust and support from people.

Wireframes Mock-up

I used Adobe Experience Design to mock up prototype invited 3 people to join the user testing. During the testing, I found some issue.

User Testing

Users could not find the unit

The penguins are calculated by pairs. However the user did not notice the unit, I changed to when hovering the bar will show the number and unit.

User could not find the location details easily

More than two steps to get the location details and only available after the user clicked booking icon. I changed this as hovering the penguin icons which swinging automatically that will show the location details.
